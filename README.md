#### 介绍
基于SSM的音乐网站+论文示例参考.系统角色：管理员、普通用户
功能模块：管理员（用户管理、音乐标签管理、付费音乐管理、订单管理、付费音频管理、系统管理等）、普通用户（个人中心、订单中心、付费音乐、收藏管理等）
技术栈：SSM，vue，layui等
测试环境：idea2024，Maven3，MySQL5.7，jdk1.8
